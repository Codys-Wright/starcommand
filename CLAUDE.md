# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository.

## Critical Rules

- **NEVER edit `flake.nix` directly** - It is auto-generated by flake-file. Edit the source modules (typically `modules/flake/dendritic.nix`) and run `nix run .#write-flake` to regenerate it.

## Build/Deploy Commands

- `just deploy` - Deploy to starcommand via deploy-rs
- `just build` - Build the starcommand configuration
- `just test` - Run flake checks
- `just fmt` - Format with nixfmt
- `just ssh` - SSH into starcommand

## Architecture

This is a dedicated flake for the starcommand self-hosting server, using:
- **den**: Aspect-based configuration
- **selfhostblocks**: Self-hosting service modules
- **flake-file**: Auto-generates flake.nix from dendritic.nix
- **deploy-rs**: Remote deployment

## Directory Structure

- `hosts/starcommand/` - Host configuration
- `modules/flake/` - Flake infrastructure (dendritic.nix, deploy-rs.nix, etc.)
- `modules/selfhost/` - Self-hosting service configurations
- `users/starcommand/` - Service user configuration
